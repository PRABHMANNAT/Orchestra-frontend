# Codebase Overview Notes

## What This Page Is

The codebase overview is the orientation layer: repos, owners, key decisions, and contested questions for the Northstar engineering surface.

It is intentionally table-stakes. GitHub can show repos, activity, and ownership hints, so this page stays modest and acts as a hub into the deeper Orchestra surfaces.

## What Links Out

- Repo cards link to light repo detail routes at `/projects/:id/overview/repos/:repoId`.
- People link to continuity profiles at `/projects/:id/memory/person/:personId`.
- Decisions and contested questions link to the context layer/rationale trace anchors under `/projects/:id/live-doc#...`.

## Differentiated Surface

The differentiated value is not this summary. It is the linked rationale trace: why a decision exists, what contradicted it, and which source proved the current truth.

This overview does one thing Git/GitHub does not do cleanly: it puts decisions beside the repos and people they affect, then links each decision to its rationale.
